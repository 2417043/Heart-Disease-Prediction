#  Heart Disease Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict the presence of heart disease using patient medical data and machine learning techniques. The dataset contains various health-related attributes such as age, blood pressure, cholesterol level, fasting blood sugar, and other clinical measurements.

The goal is to classify whether a patient is likely to have heart disease or not.

---

## 🎯 Objective

* Perform data cleaning and preprocessing.
* Handle missing values in the dataset.
* Encode categorical variables.
* Apply feature scaling using StandardScaler.
* Train and evaluate Machine Learning models.
* Compare Logistic Regression and Support Vector Machine (SVM) performance.
